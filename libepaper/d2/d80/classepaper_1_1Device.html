Detailed Description

RAII wrapper for BCM2835 device.

Manages the lifecycle of the BCM2835 library and SPI interface. Follows the RAII pattern with automatic resource cleanup.

Note
Exception Safety: Strong guarantee for all operations. Destructor is noexcept and will never throw. Move operations are noexcept.
Examples
/mnt/nas/libepaper2/include/epaper/display.hpp.

◆ init()

auto epaper::Device::init ( ) -> std::expected<void, Error>

Initialize the BCM2835 library and SPI.

This must be called before any other operations on the device. Idempotent - calling init() on an already initialized device succeeds.

Returns
void on success, Error on failure
Note
Exception Safety: Strong guarantee - if initialization fails, the device remains in an uninitialized state.
