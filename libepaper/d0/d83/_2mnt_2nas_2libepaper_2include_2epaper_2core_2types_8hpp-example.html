Color representation for e-paper displays.

Color representation for e-paper displays.Defines logical colors supported across different display modes. Not all colors are available on all displays - availability depends on the DisplayMode (see Display::available_colors()).

Note
Color values are not device-native encodings - they are converted to display-specific bit patterns by ColorManager.
// Basic color usage
display.set_pixel(10, 20, Color::Black);
display.clear(Color::White);
// Check available colors for current mode
auto colors = display.available_colors();
if (std::find(colors.begin(), colors.end(), Color::Red) != colors.end()) {
display.draw_line({0, 0}, {100, 100}, Color::Red);
}
See also
ColorManager, DisplayMode, Display::available_colors()
