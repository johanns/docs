Detailed Description

Abstract driver interface for e-paper displays.

Defines the interface that all e-paper display drivers must implement. Provides hardware abstraction for different display models.

Note
Exception Safety: All virtual methods provide basic exception safety. Derived classes must maintain object validity even if operations fail.
